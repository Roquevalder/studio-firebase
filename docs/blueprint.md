# **App Name**: Firebase Lister

## Core Features:

- Firebase Login: Authenticate with the existing Firebase database 'GestaoDeFrequencia' using appropriate credentials.
- Collection Listing: Retrieve a list of all collection names within the Firebase database.
- Document Counting: For each collection, count the number of documents it contains.
- Data Display: Present the collection names and their document counts in an organized manner within the app.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5), almost white, to ensure readability and a clean interface.
- Accent color: Indigo (#5C6BC0), a slightly lighter blue analogous to the primary color, used for interactive elements and highlights.
- Font: 'Inter', a sans-serif font, will be used for both headlines and body text, ensuring a modern and readable design.
- Use simple, outline-style icons to represent collections and documents, maintaining a clean aesthetic.
- Display collection names and document counts in a clear, tabular format for easy comprehension.
- Subtle animations for loading data and transitioning between sections to enhance user experience.
